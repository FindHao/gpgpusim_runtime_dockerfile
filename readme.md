# GPGPU-SIM Runtime Dockerfile

This is a dockerfile of runtime gpgpusim.

You can download this dockerfile and 

```shell
docker build -t gpgpusim_runtime .
```

to build a new images.

Or you can just 

```shell
docker pull findhao/gpgpusim_runtime
```
